Q2. 파이썬 코딩을 하기에 앞서 하드웨어를 이해하는 것이 중요하다고 했습니다.
    하드웨어 아키텍쳐에서 CPU와 Main Memory 그리고 Secondary Memory의 역할을
    간단하게 정리하여 봅시다.
>> CPU - 모든 컴퓨터의 명령을 연산,처리,제어함. 빠름
   Main Memory (RAM) - CPU의 동작에 필요한 내용을 기억하고 있는 주기억장치. 동일한 속도로 읽고 쓰기가 가능. 휘발성 메모리이므로 전원이 꺼지면 지워짐.
   Secondary Memory(HDD) - 물리적으로 디스크를 회전시켜 대량의 데이터를 저장할수 있는 메모리
